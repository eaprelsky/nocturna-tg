# Features Documentation

Эта директория содержит детальное описание функциональных возможностей бота.

## Доступные документы

- [biwheel-transit-charts.md](biwheel-transit-charts.md) - Полная документация биколесных карт транзитов
- [biwheel-transit-implementation.md](biwheel-transit-implementation.md) - Краткое описание реализации

## Назначение

Документы в этой директории описывают:
- Как работает функция
- Архитектура и компоненты
- API эндпоинты
- Конфигурация
- Примеры использования
- Зависимости

## Добавление новых документов

При добавлении новых функций:
1. Создайте файл с названием `feature-name.md` (lowercase, через дефис)
2. Используйте существующие документы как шаблон
3. Обновите этот README со ссылкой на новый документ
4. Добавьте ссылку в [docs/index.md](../index.md)
