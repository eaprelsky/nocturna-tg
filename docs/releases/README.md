# Release Notes

Эта директория содержит детальные release notes для каждой версии.

## Версии

- [v1.0.0](v1.0.0.md) - Первый production релиз (2025-11-21)

## Формат

Каждый release note включает:
- Дату релиза и статус
- Основные возможности
- Выполненную очистку
- Структуру проекта
- Быстрый старт
- Checklist для деплоя
- Roadmap

## Naming Convention

Файлы именуются по версии: `v{major}.{minor}.{patch}.md`

Например:
- `v1.0.0.md` - Major release
- `v1.1.0.md` - Minor release с новыми фичами
- `v1.0.1.md` - Patch release с багфиксами

Для общей истории изменений используйте основной [CHANGELOG.md](../../CHANGELOG.md).
